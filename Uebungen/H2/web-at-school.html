Web programming in a nutshell (at school)

Even though most schools teach some basic HTML/CSS, most of them fail to teach usable web skills. Even the most legacy sites have reached the conclusion that web development without a framework is barely usable, schools still teach basic web skills, that cannot be used in any company environment.

A Better option would be to have a fundamental introduction into web technologies at the core, but pivot after a day or two into a framework, that is actually used by professionals. There is a reason why all big corporations and professionals are using frameworks and always recommend to use one.
